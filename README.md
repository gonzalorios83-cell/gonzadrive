# Punto Smart OS v1

Maqueta funcional estática para GitHub Pages.

## Incluye
- Diseño oscuro/neón responsive.
- Íconos agrupados por secciones.
- Carpetas para trámites, bancos, noticias y proveedores.
- 5 accesos personalizables guardados en el navegador con localStorage.
- Buscador.
- Modo edición y arrastre simple entre accesos superiores.
- Banners publicitarios.

## Subida a GitHub Pages
Subir `index.html`, `styles.css` y `script.js` a la raíz del repositorio.
